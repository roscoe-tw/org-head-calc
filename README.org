* 專案簡介
本文件採用 **Docs-as-Code** 工作流，確保工程計算與技術文件的一致性。
- *Org-mode*: 結構化內容與代碼塊管理
- *Maxima*: 符號與數值工程計算引擎
- *Reveal.js*: CI/CD 自動化生成的 HTML5 簡報

使用 org 檔案設定碟形端板參數，上傳到 github 後由 GitHub Actions 產生結果。
