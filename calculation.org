* 厚度計算流程
** 係數 M 之計算
根據 ASME Section VIII, Div. 1, Appendix 1-4：
\[ M = \frac{1}{4} \left( 3 + \sqrt{\frac{L}{r}} \right) \]

#+NAME: calc_m
#+BEGIN_SRC maxima :var params=design_params :exports results :results html :eval yes
/* 提取參數 */
L : float(find_val("冠部內半徑", params))$
r : float(find_val("轉角內半徑", params))$

/* 檢查參數有效性 */
if (L <= 0 or r <= 0) then
    error("參數錯誤：L 或 r <= 0")$
    
/* 計算係數 M */
M_val : 0.25 * (3 + sqrt(L/r))$

/* 回傳 HTML 結果 */

** 最小需求厚度 (t)
公式如下：
\[ t = \frac{P \cdot L \cdot M}{2SE - 0.2P} \]

#+NAME: calc_t
#+BEGIN_SRC maxima :var params=design_params :exports results :results html :eval yes
/* 提取必要參數 */
P : float(find_val("設計壓力", params))$
L : float(find_val("冠部內半徑", params))$
r : float(find_val("轉角內半徑", params))$
S : float(find_val("材料許用應力", params))$
E : float(find_val("銲道效率", params))$

/* 檢查參數有效性 */
if (P <= 0 or S <= 0 or E <= 0) then
    error("參數錯誤：P, S, E 必須 > 0")$

/* 計算 M */
M : 0.25 * (3 + sqrt(L/r))$

/* 計算需求厚度 */
t_req : (P * L * M) / (2*S*E - 0.2*P)$

/* 回傳 HTML 結果 */
concat("<div id='result-t' class='results-box'><strong>(2) 需求厚度計算結果</strong><br/>",
       "t_req = <strong>", string(t_req), "</strong> mm</div>")$
#+END_SRC
