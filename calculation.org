* 封頭幾何係數 M 的計算公式
#+REVEAL_HTML: <div>
#+ATTR_HTML: :width 400px :alt "Torispherica drawing"
[[file+emacs:assets/images/Torispherical-drawing-1.png]]
#+begin_notes
圖中標示了 R (相當於 L) 與 r (Knuckle radius) 的幾何關係，這決定了應力集中係數 M。
#+end_notes

根據 ASME BPVC Section VIII, Division 1, Mandatory Appendix 1-4，
對於碟形封頭 (Torispherical Head)，其幾何係數 $M$ 的計算公式如下：

\[
M = \frac{1}{4} \left( 3 + \sqrt{\frac{L}{r}} \right)
\]

其中：
- *L* :: 封頭內部的球面半徑 (Inside Dish Radius)
- *r* :: 封頭邊緣的轉角半徑 (Inside Knuckle Radius)
#+REVEAL_HTML: </div>

** 最小需求厚度 (t)
公式如下：
\[ t = \frac{P \cdot L \cdot M}{2SE - 0.2P} \]

  fpprintprec: 5$

/* 2. 提取計算參數 */
P : float(find_val_sym("P", params))$
L : float(find_val_sym("L", params))$
r : float(find_val_sym("r", params))$
S : float(find_val_sym("S", params))$
E : float(find_val_sym("E", params))$
tn : float(find_val_sym("tn", params))$
CA : float(find_val_sym("CA", params))$

/* 3. 計算需求厚度 t_req */
M_val : 0.25 * (3 + sqrt(L/r))$
t_req : (P * L * M_val) / (2*S*E - 0.2*P)$

/* 4. 計算有效厚度 t_available */
/* 考慮公稱厚度扣除腐蝕裕度 */
t_available : tn - CA$

/* 5. 邏輯判定 */
is_pass : if t_available >= t_req then true else false$
color : if is_pass then "#4CAF50" else "#f44336"$ /* 綠色或紅色 */

/* 6. 輸出 HTML */
print(concat("<div class='results-box' style='border: 2px solid ", color, "; background: rgba(255,255,255,0.05);'>",
             "<strong style='color:", color, "; font-size: 1.2em;'>[ ", if is_pass then "PASS 符合規範" else "FAIL 厚度不足", " ]</strong><br/><br/>",
             "● 最小需求厚度 (t_req)：<strong>", string(t_req), "</strong> mm<br/>",
             "● 選用有效厚度 (tn - CA)：<strong>", string(tn), " - ", string(CA), " = ", string(t_available), "</strong> mm<br/>",
